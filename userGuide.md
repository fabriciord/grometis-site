# Guia do Usuário - GrOMEtiS Software & Integration

## Informações do Site

**Propósito:** Site institucional da GrOMEtiS Software & Integration apresentando serviços de desenvolvimento de software terceirizado, integração de sistemas e análise de métricas.

**Acesso:** Público - não requer login

## Powered by Manus

**Stack Tecnológico:**
- Frontend: React 19 + TypeScript + Tailwind CSS 4 + Shadcn/UI
- Animações: Framer Motion para transições suaves
- Ícones: Lucide React
- Deployment: Auto-scaling infrastructure with global CDN

O site foi desenvolvido com tecnologias modernas que garantem performance excepcional, design responsivo e experiência de usuário fluida em todos os dispositivos.

## Usando Seu Website

O site da GrOMEtiS apresenta de forma clara e profissional os serviços oferecidos pela empresa. A navegação é intuitiva com menu fixo no topo permitindo acesso rápido às seções principais.

Na página inicial você encontra apresentação destacada da empresa com call-to-actions para "Conheça Nossos Serviços" e "Solicitar Orçamento". Role para baixo para ver os 4 serviços principais: Desenvolvimento de Software, Integração de Sistemas, Métricas e Analytics, e Consultoria Técnica.

A seção "Sobre" explica o significado do nome GrOMEtiS (Gro = Group, Met = Metrics, iS = integration System) e destaca os benefícios da terceirização. Badges de tecnologias mostram as competências técnicas da empresa.

Na seção de contato você encontra email, telefone e localização da empresa, além de botão para solicitar orçamento gratuito.

## Gerenciando Seu Website

Use o painel de gerenciamento do Manus para:
- **Settings → General**: Alterar título e logo do site
- **Settings → Domains**: Configurar domínio personalizado
- **Dashboard**: Monitorar visitas e analytics
- **Code**: Baixar ou editar arquivos do projeto

Para modificar conteúdo do site, edite o arquivo `client/src/pages/Home.tsx` e solicite ajuda ao Manus AI para implementar mudanças.

Converse com o Manus AI a qualquer momento para solicitar alterações ou adicionar funcionalidades. Você pode expandir o site adicionando blog, portfólio de projetos, formulário de contato funcional ou integração com CRM.
